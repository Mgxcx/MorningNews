# MorningNews
Application React créée lors de la 6ème semaine de formation du 16 au 20 Novembre. Application qui pioche dans une API de news : les articles des presses anglaises ou françaises (au choix). Nous pouvons ajouter des articles dans notre "wishlist" et les supprimer une fois que nous les avons consultés.
